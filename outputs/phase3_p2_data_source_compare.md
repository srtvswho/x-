## 1. Stooq.com

❌ **沙箱不可用**

```
curl -s 'https://stooq.com/q/d/l/?s=aapl.us&i=d'
→ HTTP 200, 但返回 HTML 含 JS challenge:
  <script nonce=...>
  (async()=>{const c=..., d=4, ...
  ...requires JavaScript to verify your browser...})();
  </script>
```

**根因**:Stooq 用 JavaScript-based 反爬(浏览器验证 cookie),curl/requests 不执行 JS,被反爬识别为 bot 拦截。

**绕过办法**(都不能用):
- 浏览器自动化(playwright/selenium):沙箱没有 GUI
- JS 引擎(pyexecjs):需要先解密 SHA-256 challenge,每次 nonce 不同,不可行
- 第三方代理:无免费可用

**结论**:Stooq 在沙箱跑不通,必须换源。
